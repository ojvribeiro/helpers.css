Text alignment

You can use these classes to easily align the texts of you page. The alignment depends on the element the text is within.

chevron_right
text-left

Aligns text to the left.

<p class="text-left">Left aligned</p>

Left aligned

The default value to text-align is already left, so you should only use it to force the left alignment.

chevron_right
text-right

Aligns text to the right.

<p class="text-right">Right aligned</p>

Right aligned

chevron_right
text-center

Aligns text to the center (horizontally).

<p class="text-center">Center aligned</p>

Center aligned

chevron_right
text-middle

Aligns text to the middle (vertically) of the parent element. I added some artificial height of 200px for better visualization.

<div class="text-middle" style="height: 200px;">Middle aligned</div>
Middle aligned

You can also use it combined with text-center to centralize the text inside a parent.

<div class="text-middle text-center" style="height: 200px;">Middle aligned</div>
Middle aligned

Feel free to combine it with text-left or text-right too.

Text size

Text sizes are well defined, so you have consistance across your site/app. Because we use em instead of px, texts are fluid for every screen resolution.

chevron_right
text-smaller

The text-smaller class has a size of .7em. It is really tiny, so you should use it only in specific cases.

You should not abuse of this size or the user experience will be seriously affected. 

<p class="text-smaller">Sample text</p>

Sample text

chevron_right
text-small

The text-small class has a size of .9em. You can use it for small buttons or footer notes, for example. 

<p class="text-small">Sample text</p>

Sample text

chevron_right
text-medium

The text-medium class has a size of 1em. This is the default size for using in an article/blog post. 

<p class="text-medium">Sample text</p>

Sample text

chevron_right
text-large

The text-large class has a size of 2em. It has the equivalent size of a h2 tag, but you should use it within the h2 to keep semantics. 

<h2 class="text-large">Sample text</h2>

Sample text

chevron_right
text-larger

The text-larger class has a size of 3em. It has the equivalent size of a h1 tag, but, just like text-large, you should use it within the h1 to keep semantics. 

<h1 class="text-larger">Sample text</h1>

Sample text
