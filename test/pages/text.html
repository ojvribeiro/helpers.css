Text alignment

You can use these classes to easily align the texts of you page. The alignment depends on the element the text is within.

chevron_right
text-left

Aligns text to the left.

<p class="text-left">Left aligned</p>

Left aligned

The default value to text-align is already left, so you should only use it to force the left alignment.

chevron_right
text-right

Aligns text to the right.

<p class="text-right">Right aligned</p>

Right aligned

chevron_right
text-center

Aligns text to the center (horizontally).

<p class="text-center">Center aligned</p>

Center aligned

chevron_right
text-middle

Aligns text to the middle (vertically) of the parent element. I added some artificial height of 200px for better visualization.

<div class="text-middle" style="height: 200px;">Middle aligned</div>
Middle aligned

You can also use it combined with text-center to centralize the text inside a parent.

<div class="text-middle text-center" style="height: 200px;">Middle aligned</div>
Middle aligned

Feel free to combine it with text-left or text-right too.

chevron_right
text-justify

Makes the text justified.

Font size

Text sizes in Helpers.css follows well-defined standards, keeping consistency across your project.

There are six text sizes you can use:

  • Largest
  • Larger
  • Large
  • Medium
  • Small
  • Smaller

chevron_right
b:fs-xs

The b:fs-xs class has a text size equivalent to 10px. It is really tiny, so you should use it only in very specific cases.

You should not abuse of this size or the user experience will be seriously affected. 

<p class="b:fs-xs">Sample text</p>

Sample text

chevron_right
b:fs-sm

The b:fs-sm class has a text size equivalent to 16px and it's the default text size if none is defined. 

<p class="b:fs-sm">Sample text</p>

Sample text

chevron_right
b:fs-md

The b:fs-md class has a text size equivalent to 21px. It is recommended that you use this class in paragraphs and subtitles. 

<p class="b:fs-md">Sample text</p>

Sample text

chevron_right
b:fs-lg

The b:fs-lg class has a text size equivalent to 32px. You should use it within the h3 tag when necessary to keep semantics. 

<h3 class="b:fs-lg">Sample text</h3>

Sample text

chevron_right
b:fs-xl

The b:fs-xl class has a size of 42px. You should use it within the h2 tag when necessary to keep semantics. 

<h2 class="b:fs-xl">Sample text</h2>

Sample text

chevron_right
b:fs-xxl

The b:fs-xxl class has a size of 56px. You should use it within the h1 tag when necessary to keep semantics. 

<h1 class="b:fs-xxl">Sample text</h1>

Sample text
